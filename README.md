# 💰 Financial Dashboard

A full-stack **Financial Dashboard** that helps users track and analyze their financial data through interactive charts and real-time statistics. The application leverages the **MongoDB Aggregation Pipeline** to calculate financial summaries and **Recharts** for data visualization. Complex state management is handled using **useReducer**.

## 🚀 Features

### 📊 Dashboard Features

* View total income, expenses, and balance
* Interactive financial charts and analytics
* Category-wise expense breakdown
* Real-time financial summaries
* Responsive and user-friendly dashboard
* Filter and analyze financial data

### 📈 Data Visualization

* Built with **Recharts**
* Bar charts, line charts, and pie charts
* Visual representation of financial trends
* Easy-to-understand analytics and reports

### ⚡ Data Processing

* **MongoDB Aggregation Pipeline** for:

  * Calculating total income and expenses
  * Generating category-wise summaries
  * Computing balances and financial statistics
  * Efficient server-side data processing

### 🧠 State Management

* **useReducer** for handling complex state logic
* Predictable state transitions
* Centralized management of filters, transactions, and dashboard data
